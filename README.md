# AI Creative Studio - HeroUI v3 Showcase

A beautiful, modern demo application showcasing the power and flexibility of **HeroUI v3** (formerly NextUI) components.

## 🌟 Features

This demo application includes 4 fully-designed pages:

1. **Landing Page** - Hero sections, pricing cards, feature highlights with smooth animations
2. **Dashboard** - Data tables, activity feeds, statistics cards, progress indicators, loading states
3. **Forms & Inputs** - Date pickers, color pickers, sliders, switches, accordions, validation
4. **Components Gallery** - Interactive showcase of 30+ HeroUI components including:
   - Navigation bars and menus
   - Buttons (all variants and sizes)
   - Dropdowns and action menus
   - Modal dialogs
   - Tabs and pagination
   - Avatars and badges
   - Tooltips
   - Keyboard shortcuts (Kbd)

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Pages Overview

### Landing Page (`/`)
A marketing landing page with gradient backgrounds, animated feature cards, and pricing tiers.

### Dashboard (`/dashboard`)
A comprehensive admin dashboard featuring:
- Real-time statistics with trend indicators
- Interactive data tables with avatars and progress
- Activity feed with user actions
- Loading states demonstration

### Forms (`/forms`)
Extensive form components showcase:
- Input fields with icons and variants
- Date and time pickers
- Date range selection
- Sliders with step markers
- Checkboxes and radio buttons
- Switches and toggles
- Select dropdowns
- Accordion FAQ section

### Components Gallery (`/components`)
Interactive component library covering:
- Responsive navigation bars
- All button variants (solid, bordered, flat, faded, shadow, ghost)
- Dropdown menus with icons
- Modal dialogs
- Tabbed interfaces
- Avatar groups with badges
- Tooltips (all placements)
- Pagination controls
- Keyboard shortcuts display
